Tu Editor de Código (VS Code o Cursor)

Para trabajar de forma eficiente, necesitas un editor de código instalado en tu computadora. Puedes elegir entre estas dos excelentes opciones:

Opción 1: Visual Studio Code (VS Code)

Es el editor más popular y cuenta con un soporte excelente para lectores de pantalla.

  • En Windows: Descarga el instalador ejecutable desde el sitio oficial de VS Code e instala de forma tradicional, o ejecuta en tu consola: winget install Microsoft.VisualStudioCode
  • En Mac: Descarga el archivo ZIP desde la web oficial, descomprímelo y arrastra la aplicación a tu carpeta de Aplicaciones, o usa tu terminal con Homebrew ejecutando: brew install --cask visual-studio-code

Opción 2: Cursor (IDE Agéntico)

Es un editor moderno basado en VS Code que integra herramientas potentes de Inteligencia Artificial de forma nativa.

  • Puedes descargarlo directamente desde el sitio oficial de Cursor (cursor.com) para tu sistema operativo y seguir el asistente de instalación estándar, el cual te permitirá importar toda tu configuración previa de VS Code si ya lo tenías instalado.

Además, estoy profundizando en la accesibilidad al programar siendo ciego. Aprender a hacer código accesible es indispensable y mejora la experiencia para todos los usuarios.
